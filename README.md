# Objective
The objective of this project is to analyze long-term trends in agricultural production data to identify patterns and regional differences over time.

# Dataset
-European grape yield data (2000–2023)
-Source: FAOSTAT
-Multi-country time series dataset

# Tools & Technologies
-R (ggplot2, dplyr, readr)
-Data visualization and transformation

# Workflow
-Data import and cleaning
-Data transformation and aggregation
-Exploratory data analysis (EDA)
-Time series visualization
-Comparative analysis across countries

# Key Results
-Identified long-term trends in production across regions
-Highlighted top-performing countries based on yield
-Observed variations in production patterns over time

# Key Insights
-Agricultural production shows clear temporal and regional variability
-Some countries consistently outperform others in yield
-Visualization enables effective comparison across multiple dimensions

# Project Outputs
-Regression plot of yield trends
-Top countries comparison
-Average yield trend visualization
